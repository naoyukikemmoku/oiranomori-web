# ライトフォレスト 実装用リファレンス（2026.04版）

## 1. キャッチコピー・導入文

* **メインタイトル**: おいらが家族と森の中に住んでいる管理人常駐の安心・安全なキャンプ場
* **サブタイトル**: 里山の管理された広葉樹林をキャンプを通じて感じてください
* **本文**: 日当たり良好、眺め良し、里山に囲まれた静かで明るいライトフォレストのキャンプサイト。全面焚き火可能、ファミリー向けオートサイト、プライベート感・野営感たっぷり軍幕やブッシュクラフトの似合う直火可能なソロキャンパーのためのソロ区画、ソログル、ファミリーグループも安心広々オートサイト、ノーリード可能な超広いドッグランサイトなど、多様なスタイルに対応。森の木々を使ったタープ設営やハンモック泊も容易。
* **四季の魅力**: 春の新緑、夏の昆虫、秋の紅葉、冬の陽だまり。四季の移り変わりをダイナミックに感じられる環境。

## 2. キャンプサイトの種類と特徴

### サイトタイプの定義

* **オートサイト**: 区画内に車を乗り入れ・横付け可能。有効面積は車の面積を除いた広さ。
* **区画サイト**: 車の乗り入れができないサイト。サイト付近に駐車スペースあり。
* **ドッグランサイト**: 柵内でキャンプ可能。車乗り入れ可・ノーリードOK。
  ※「区画外」は提供終了につき廃止済み。

### サイズ別目安表（有効面積＝駐車スペースを除く）

| サイズ | 有効面積 | 収容・レイアウト目安 |
| :---- | :---- | :---- |
| **XL** | 110㎡以上 | 10人以上：超大型テント＋タープ、車3台＋ソロテント3張り |
| **LL** | 80～109㎡ | 6〜8人：超大型テント1張り＋タープ、ソロテント数張り |
| **L** | 60～79㎡ | 4〜6人：大型テント1〜2張り＋タープ |
| **M** | 40～59㎡ | 2〜4人：中型テント1張り＋タープ |
| **S** | 20～39㎡ | 2人以下：小型テント1張り＋タープ |
| **SS** | 20㎡未満 | 2人以下：ソロテント1張り（通称ソロ区画）。プライベート感重視。 |

## 3. 施設案内・スペック

* **受付・売店**: 9:00〜17:30。飲料（水・ビール等）、薪の販売。
* **管理棟**: 旧管理棟は現在利用停止中。
* **トイレ（新築）**: 水洗トイレ。洋式・男女各2室完備。
* **水道・流し**: **給湯器付き**。すべて水道水（飲用可）。※洗剤・油・残飯は流さないこと。
* **通信環境**: 無料Wi-Fiあり（接続詳細は場内掲示）。**※ソロ区画、オートA、オートBはエリア外のため接続不可**。
* **AC電源**: 有料オプション。消費電力 **600Wまで**。ブレーカー落ちはセルフ復旧。
* **駐車場**: オートサイト以外に15台分（1台 2,200円）。

## 4. 利用ルールとタイムスケジュール

### タイムスケジュール

* **09:00**: 受付オープン / アーリーチェックイン開始
* **12:00**: 宿泊チェックアウト / 通常チェックイン開始
* **17:30**: 売店・薪販売終了
* **18:00**: 外出門限（町内放送） / 日帰りチェックアウト。門限以降の車移動禁止。
* **21:00**: 星空タイム。外への光漏れ禁止、談笑禁止、車のドア開閉注意。
* **23:00**: 消灯・完全サイレント（無言の焚き火のみOK）。
* **06:00**: 車の移動・朝バトニング開始OK。

### 重要ルール

* **チェックアウト手続き**: 帰宅時はHPまたはLINEから手続きを行うこと。
* **焚火処理**: 灰は指定ペール缶へ。蚊取線香、アルミ、ゴミ、タバコ等の混入厳禁。蓋は必ず閉める。
* **ゴミ**: 灰以外は全て持ち帰り。
* **ペット**: ドッグラン以外は要リード。ふんの処理、管理は飼い主の責任。
* **禁止・違法事項**: スピーカー、アイドリング、打ち上げ花火、発電機、21時以降の談笑。樹木の損傷や窃盗、不法投棄は即警察へ通報。
* **免責**: 瑕疵によるものを除き、事故・怪我は自己責任。
