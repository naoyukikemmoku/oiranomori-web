## 3. メニュー・UI構成

### 基本方針
来場者の行動パターン優先順：**初めての人（A）> リピーター（B）> 迷っている人（C）**
「初めての人が迷わず予約に至る」ことを最優先に設計する。

### ① 常時表示（どこにいても見える）
| 要素 | 位置 | リンク先 |
|---|---|---|
| 予約する | 下部中央フローティング | ProX直通（フィールド別） |
| ライト / ディープ切り替え | 上部固定 | `/` または `/deep/` |
| 天気ウィジェット | 上部右寄せ | `/now/#weather` |

### ② ハンバーガーメニュー（右上固定）
初めての人が知りたい順に配置する。

```
予約する → ProX直通
─────────────────
ライトフォレスト 利用案内 → /light-guide/
ディープフォレスト 利用案内 → /deep-guide/
アクセス・場内マップ → /access/
─────────────────
お知らせ → /news/
よくある質問 → /faq/
薪・レンタル → /firewood/
回数券・パスポート → /passport/
お問い合わせ → /contact/
─────────────────
予約ガイド → /reservation-guide/
チェックアウト連絡 → /checkout/
会員ログイン → ProX
新規会員登録 → ProX
─────────────────
森の物語 → note
```

### ③ 右サイドフローティング
- 縦書き「利用案内」→ `/light-guide/` または `/deep-guide/`（フィールド連動・将来対応）

### ④ フッター（全ページ共通）
- キャンセルポリシー → `/policy/#cancellation`
- 利用規約 → `/policy/#terms`
- プライバシーポリシー → `/policy/#privacy`

### Note移管コンテンツ（HPから削除）
以下はHPには載せない。noteで管理する。
- 焚火のすすめ・過ごし方の読み物
- 開拓日誌
- 生きもの観察記録
- おすすめキャンプ場

## 4. おいらの森 固有仕様

**料金**
5段階シーズナリティ（S1〜S5）。HP上ではサイト種別ごとの最低料金を「辞書」として提示し、詳細・予約はProXへ誘導する。

**マップ**
- アクセス用: 広域の静的マップ（`/access/`）
- 滞在者用: 現在地連動の動的マップ（Google Maps Platform）
- 両者は別コンポーネントとして分離する

**プランギャラリー**
- 各フィールドLPに3〜5プランを「写真＋キャッチコピー」カード形式で表示
- 画像クリックでProXの予約画面へ直接遷移
- 全プラン一覧は `/experience/#plans` に設置（比較表形式）

**予約ガイド**
「会員特典（ポイント）」と「買い物カゴ機能（複数予約・複数日程の同時予約）」のメリットを明示的に訴求する。

## 5. 現行HPコンテンツの引き継ぎ方針

| コンテンツ | 対応 |
|---|---|
| クイックガイド（場所・アクセス・設備・ルール） | そのまま使える。新HPで再利用 |
| プラン一覧（7プラン） | 再構成が必要。ProXリンクと写真を整備 |
| ライト/ディープ フィールド説明文 | 文章を磨いて再利用 |
| お知らせ | microCMS管理。継続 |
| イベント情報 | microCMS管理。継続 |
| ソロキャンプの勧め・焚火のすすめ | Note移管 |
| オイラーズ・農業体験 | Phase 2以降で検討 |
| 開拓日誌・生きもの観察記録 | Note移管 |
