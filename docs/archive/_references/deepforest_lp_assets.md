# Deep Forest: ランディングページ（LP）用アセット

## 1. コンセプト & コピー
* **コンセプト**: 樹齢80年のスギ・ヒノキが創り出す、圧倒的な静寂と闇
* **メインコピー**: この雰囲気は、いかないとわからない。
* **導入文**:
    太平洋戦争で荒廃した日本の森を回復させるため、国策で植えられたスギ・ヒノキたち。それから約80年が経過した今、ディープフォレストの木々は大きなもので直径50cm、樹高25mを超え、整然と、しかし圧倒的な密度でそびえ立っています。

## 2. 4つの情緒的体験（訴求ポイント）
1.  **耳がキーンとするほどの静寂**
2.  **本物の闇に包まれる夜**
3.  **変化する森の表情**
4.  **野生との境界線**

## 3. 基本の4鉄則
利用者が守るべき、この森の精神的基盤。
1.  **森の静寂と闇夜、自然の音を楽しむこと。**
2.  **現地で使用可能なものは「落葉・落枝」のみ。**
3.  **利用後は現状回復し、足跡以外の痕跡を残さないこと。**
4.  **トイレ以外の場所での排泄（野外排泄）は厳禁。**

## 4. ご利用ワンポイント（重要事項）
ユーザーに「一般的なキャンプ場との違い」を認識させ、期待値を調整するための重要項目。
1.  **異質の環境**: キャンプ場としての人工的な整備をしていないため、一般的なキャンプ場とは全く異なります。
2.  **不便の許容**: 基本的に不便な場所です。趣旨を理解の上で利用してください。事前の下見を強く推奨します。
3.  **場所は非公開**: ライトフォレストから「車で約7分」の別の山脈にあります。場所は非公開のため、受付で案内を受けてください。
4.  **無人管理**: ディープフォレストに管理人は常駐しません。受付はライトフォレストです。
5.  **静寂の遵守**: グルキャンで騒いだり、酒宴を楽しむ場所ではありません。お静かにお過ごしください。
6.  **足元の備え**: 雨後はぬかるみやすいため、長靴の携行を推奨します。
7.  **装備の選択**: 広場から先は車両進入禁止です。傾斜がありキャリーカーの使用は困難なため、バックパックが便利です。
