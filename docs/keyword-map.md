# おいらの森 キーワードマップ

> 用途：現状サイトが「拾えてる検索意図」を6柱で逆算し、狙いとの差分＝未成熟箇所を炙り出す**ギャップ検出器**。
> 置き場候補：`oiranomori-web/docs/keyword-map.md`
> 作成：2026-06-19 / 現物（oiranomori.jp）を fetch して作成。カバレッジは現状推定を含む。

---

## 0. 前提

- **6柱は「ページ」ではなく「テーマ」。** 柱はLight/Deepを横断する。柱＝ページで作ると意図がページを奪い合う（カニバリ）。
- **柱の落とし先は3択：**
  - **(a) 拡充** … 既存ピラー（`/`＝Light / `/deep/` / 各guide / `/access/`）の本文を厚くする
  - **(b) ガイド記事** … 情報型で検索され中身が濃い柱だけ、独立記事を1本立てる（本文末から料金/予約へ落とす）
  - **(LP)** … スタンドアロンLPは「料金/予約」だけ。例外候補は「別格」欄で要判断
- **狙いは単独語でなく複合語（体験 × 地理 × 客層/条件）の束。** 地理アンカー：那須烏山 / 栃木 / 北関東 / 関東 / 東京から。
- **`/sites/`（区画紹介）レイヤーの新設**により、客層・スタイル・人数・設備の検索意図は `/sites/` で受ける。homepage / guide は体験テーマ（柱1〜3）とフィールド世界観に専念し、客層意図を背負わない。
- カバレッジ：◎受けてる ○一応 △弱い ×無し

## 4. ギャップ集約（アクション）

### A. 別格（取引型・希少差別化）
- **犬/ドッグラン** … `/sites/dogrun/` 独立1ページで受ける（検索熱と差別化のため客層フィルタの例外として独立URL化）。「ペット可 キャンプ場 栃木 / 貸切ドッグラン」を受ける。
- **電車アクセス** … `/access/` に電車アクセスの節＋h2を足す（据え置き。スタイルでなくアクセス形態のため `/sites/` には移さない）。独立ページは作らない。
- **制約**：いずれも「埋もれた1行」でなく**固有h2＋本文depth**を持たせる。買い足し程度だと拡充にならず意図を受けられない。

### B. 新規ガイド記事（情報型・じわじわ効く・(b)）
- 食材（やつだの地のもの・きのこ・地元食）
- 星空
- 生きもの・野鳥
→ 3本とも柱の「×」。情報型ロングテール＋差別化。本文末から料金/予約へ落とす。

### C. 拡充（(a)・既存を厚く）
- 柱3 四季（星・生物への橋渡し）/ 柱4 Deep（ブッシュクラフト深掘り）/ 柱5 家族・ソロ / 柱6 デイ

### D. 料金/予約LP（新規・別途App側で実装中）

## 5. 注意・禁忌

- **直火**：title / meta description / JSON-LD に入れない（絶対線）。本文の体験描写のみ。
- **`/properties_features/` taxonomy**：hammock-available 等のtermページが homepage と本文重複の疑い（fetch時、本文がhomepage導入文と一致）。重複＝カニバリ＋インデックス膨張リスク。各termに固有本文が無いなら (1)`noindex`化 か (2)固有コンテンツ付与 のどちらかで整理が要る。→ **技術監査の宿題として別出し**。
- **カニバリ**：`/sites/` は homepage と別レイヤー（②③が非index）のためカニバらない。柱5の家族・ソロ・犬は `/sites/` で受ける。
